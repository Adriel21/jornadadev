# Estudo Linguagem Harbour - Módulo 01

- Linguagem compilada, multiparadigma, sucessora da Linguagem Clipper. 
- O compilador harbour.exe ou hbc traduz o código .prg para C. Esse, que enfim, é compilador com um compilador C. Portanto, ocorre um processo de transpilação. Similar ao TypeScript - para linguagem JavaScript. 
- hbmk2 se trata da ferramenta de build.
