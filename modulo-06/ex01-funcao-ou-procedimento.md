# Exercício 1 — Função ou procedimento? (conceitual)

- Str() — converte número em string. Resposta: *Função*
- QOut() — exibe texto na tela. Resposta:*Procedimento*
- Date() — retorna a data atual. Resposta: *Função*
- Len() — retorna o tamanho de uma string ou array. Resposta: *Função*
- Upper() — retorna a string em maiúsculas. Resposta: *Função*
- ClearScreen() — limpa a tela. Resposta: *Procedimento*
