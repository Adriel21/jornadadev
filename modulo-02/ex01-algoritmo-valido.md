## Isto é um algoritmo válido? 

1 - Some dois números
R: Não é um algoritmo válido, pois, falta especificação lógica para que se torne um algoritmo. Um algoritmo é uma sequência de passos que visa solucionar um problema. Para que a expressão possa ser um algoritmo válido, é necessário que seja indicado quais números devem ser utilizados para a soma e como esses números são coletados.

2 - Leia um número. Se for positivo, mostre 'positivo'. Senão, mostre 'negativo'
R: É um algoritmo válido. Especifica uma entrada. Em seguida, é feito uma validação sobre essa entrada. Por fim, se gera um resultado. É ordenado e finito. Porém, pode haver ambiguidade em relação ao 0.

3 - Fique repetindo até acabar
R: Também não é um algorítmo válido. Não há ordem. O que será repetido? O que inicia a repetição? Até acabar é ambíguo, qual o critério para finalização da repetição? 

4 - Pegue um ingrediente qualquer e cozinhe
R: Não é um algoritmo válido. Não há especificação, não há passos. É uma ordem, não um algoritmo.
